## 🌟 Características

| Módulo | Descripción |
|---|---|
| 🍽️ Platos | Administración del menú |
| 📝 Pedidos | Control de órdenes y mesas |
| 👨‍🍳 Cocina | Seguimiento de preparación |
| 💳 Facturación | Registro y cierre de cuentas |
