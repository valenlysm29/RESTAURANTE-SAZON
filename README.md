# 📖 Descripción del Proyecto

**Sazón** es un sistema web creado para mejorar la gestión y organización de los procesos internos de un restaurante.

El proyecto permite administrar diferentes áreas del negocio desde una sola plataforma, facilitando el control de información y optimizando el trabajo del personal.

La aplicación cuenta con un diseño moderno enfocado en una experiencia sencilla, rápida y atractiva para los usuarios.

El sistema busca solucionar problemas comunes dentro de un restaurante como:

- Desorden en la gestión de pedidos
- Falta de comunicación entre áreas
- Dificultad para controlar información
- Procesos manuales lentos
- Problemas en la administración de ventas

---

# 🎯 Objetivo General

Desarrollar una plataforma web que permita mejorar la administración de un restaurante mediante herramientas digitales que faciliten el control de pedidos, platos, cocina y facturación.

---

# 🎯 Objetivos Específicos

✅ Digitalizar procesos internos del restaurante.  

✅ Mejorar el control de pedidos realizados.  

✅ Facilitar la comunicación entre atención y cocina.  

✅ Mantener organizada la información del menú.  

✅ Optimizar el proceso de facturación.  

✅ Brindar una interfaz amigable y profesional.

---

# 💡 Problemática

Los restaurantes necesitan mantener una buena organización para brindar una atención eficiente.

Cuando los procesos se realizan manualmente pueden aparecer problemas como:

❌ Errores al registrar pedidos.  
❌ Pérdida de información.  
❌ Retrasos en la preparación.  
❌ Mala coordinación entre trabajadores.  
❌ Dificultad para controlar las ventas.

Por esta razón se plantea el desarrollo de **Sazón**, un sistema que permita centralizar y mejorar estos procesos.

---

# 🚀 Solución Propuesta

El sistema permite manejar un flujo organizado:

```
Cliente
   │
   ▼
Registro del pedido
   │
   ▼
Área de cocina
   │
   ▼
Preparación del plato
   │
   ▼
Entrega al cliente
   │
   ▼
Facturación
```

Con esto se logra una mejor coordinación entre las diferentes áreas del restaurante.

# 🖥️ Módulos del Sistema

## 🍽️ Gestión de Platos

Permite administrar los productos disponibles en el restaurante.

Funciones:

✔ Registrar nuevos platos  
✔ Actualizar información  
✔ Organizar categorías  
✔ Controlar disponibilidad  
✔ Mostrar menú actualizado  

---

## 📝 Gestión de Pedidos

Módulo encargado de controlar las órdenes realizadas.

Características:

✔ Registro de pedidos  
✔ Control por mesa  
✔ Seguimiento del estado  
✔ Organización de solicitudes  
✔ Mejor atención al cliente  

---

## 👨‍🍳 Panel de Cocina

Diseñado para facilitar el trabajo del área de preparación.

Permite:

✔ Visualizar pedidos pendientes  
✔ Controlar estados de preparación  
✔ Organizar tiempos de trabajo  
✔ Mejorar la coordinación en cocina  

---

## 💳 Facturación

Permite administrar el proceso de pago.

Incluye:

✔ Registro de ventas  
✔ Control de cuentas  
✔ Métodos de pago  
✔ Cálculo de impuestos  
✔ Cierre de pedidos
